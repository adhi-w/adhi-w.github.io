Oulu Campus Virtual Environments

The Oulu Campus Virtual Environment is a remarkable portfolio project that showcases the designer's creativity and expertise in the field of 3D modeling and virtual reality. The project is based on the University of Oulu Tellus and custom building, and the designer has created a stunning 3D model of the campus environment.

The model includes detailed and realistic features such as walls, floors, ceilings, pillars, windows, doors, furniture, and props. The designer has also incorporated advanced features such as PBR texture materials, lighting probes, light maps, human avatars, and character animations, which add a new level of depth and realism to the virtual environment. Moreover, the designer has implemented source codes for interactable objects and optimized the environment's performance for a smooth user experience. This project is a testament to the designer's talent and skill, as well as the potential of virtual reality technology to create immersive and engaging environments.
